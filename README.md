Fenced converts your plain text files into clean, styled web pages. You save your work as Markdown, and the tool builds a complete HTML file. This file contains your text, headers, and code sections. The results look good on any screen. Since the tool runs inside your web browser, your data stays on your computer. You keep your privacy because no information travels to a cloud server.

## 🛠️ What this tool does

Developers often write documentation in Markdown. Regular tools turn this text into basic, boring pages. Fenced makes these pages look like professional articles. It focuses on the code sections within your documents. 

Standard Markdown editors just place code inside boxes. Fenced adds themes and decorations to those boxes. It makes your code look like it belongs in a terminal window or a modern code editor. You see real text and real code. You do not get blurry screenshots that are hard to read. You get actual HTML files that you can host on any website.

## ⚙️ Using the software

When you open Fenced, you see a simple window. The interface splits into two halves. The left side is where you type or paste your Markdown text. The right side shows a live preview of how your document looks.

### Adding your Markdown
Type your document directly into the left side. Use standard Markdown symbols. For example, use `#` for main titles and `##` for sub-titles. Surround your text with dashes to create lists. 

### Styling your code
To show code, press the backtick key three times. You see the code block appear. Fenced detects the language you use. It applies colors to your code automatically to make it easy to read. You can choose from different themes in the settings menu. Themes like Catppuccin provide soft, pleasing colors that reduce eye strain.

### Exporting your work
When your document looks correct, click the "Export" button. You choose where to save the resulting HTML file. Once saved, you can open this file in any web browser like Chrome, Firefox, or Edge. Your document will display exactly as it looked in the preview window.

## 📝 Frequently asked questions

### Does this tool send my files to the internet?
No. Fenced runs entirely on your local machine. No data leaves your computer. Your files remain private.

### Can I change the color of the code boxes?
Yes. Open the settings menu to browse available themes. The tool supports many color schemes suitable for different environments.

### Is the code inside the HTML searchable?
Yes. Because we use text-based HTML rather than images, you can search, copy, and paste the code from your final documents.

### What happens if the software stops responding?
Close the application and open it again. Since the tool does not rely on complex background services, a simple restart fixes most minor issues. Ensure you save your work by clicking the "Save" button frequently.

### Can I share these files with others?
Yes. Once you export the HTML file, you can send it to anyone. They open the file in their browser. They do not need to install Fenced to view your documents.
